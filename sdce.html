Second Degree Conic Equation Analysis and Questions

Equation Analysis

The second degree conic equation is given by:

Equation: Ax2 + Bxy + Cy2 + Dx + Ey + F = 0

or it can also be written as ax2 + 2hxy + by2 + 2gx + 2fy + c = 0

Important points to note:

01. A pair of straight lines if Œî = 0 where Œî = abc + 2fgh - af2 - bg2 - ch2
or as determinant: Œî = a h g h b f g f c
02. A circle if Œî ‚â† 0, a=b and h=0
03. A parabola if Œî ‚â† 0 and h2 - ab = 0
04. An ellipse if Œî ‚â† 0 and h2 - ab < 0
05. A hyperbola if Œî ‚â† 0 and h2 - ab > 0

So, our analysis on the second degree conic equation has been completed.
